Settings

Polling & storage

Hashrate smoothing: tau (time constant) of the EMA applied server-side. 60s is a good trade-off between responsiveness and stability. Set to 0 to see raw firmware values.

Alert thresholds

Global kill-switch. When disabled, MinerWatch continues recording alerts to the database but does not send push notifications to anyone. Browser subscriptions remain unchanged.

If the condition persists (temp still above threshold, miner still offline), MinerWatch will re-emit the alert and push every "Repeat alert every" seconds (default 600 = 10 min).

Push notifications (browser)

When a threshold is exceeded or a miner goes offline, MinerWatch sends a native notification to the operating system (even with the tab closed, as long as the browser is open).

Status: checking...

"Disable (this browser)" stops receiving notifications only here. If you have subscribed other devices/tabs in the past and notifications still arrive, use "Remove ALL" to clean the subscriptions list on the server.

Compatible with Chrome, Edge, Firefox, Safari 16+. On iOS requires "Add to Home" as a PWA.
