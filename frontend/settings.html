Polling & storage

Hashrate smoothing: tau (time constant) of the EMA applied server-side. 60s is a good trade-off between responsiveness and stability. Set to 0 to see raw firmware values.

Alert thresholds

Global kill-switch. When disabled, MinerWatch continues recording alerts to the database but does not send push notifications to anyone. Browser subscriptions remain unchanged.

If the condition persists (temp still above threshold, miner still offline), MinerWatch will re-emit the alert and push every "Repeat alert every" seconds (default 600 = 10 min).

Browser push notifications

Native OS notifications delivered by the browser. Work as long as the browser is open, even with the tab closed.

When off, MinerWatch stops sending browser pushes for new alerts. Existing browser subscriptions stay registered so re-enabling works instantly.

Status: checking...

"Disable (this browser)" stops receiving notifications only here. If you have subscribed other devices/tabs in the past and notifications still arrive, use "Remove ALL" to clean the subscriptions list on the server.

Heads up: browsers expose the Web Push API only on https:// or http://localhost. If you open MinerWatch via the Mac's LAN IP from an iPhone or another device, push will appear as "not supported" — use the Telegram channel below instead.

Telegram notifications

MinerWatch can send alerts to a Telegram chat or group. Works on any device (iPhone, Android, desktop) without HTTPS, since it's the server that calls Telegram's API — not your browser.

 

How to set up — step by step
  1. Open Telegram → search for @BotFather → send /newbot and follow the prompts.
  2. Copy the token BotFather gives you, paste it in Bot token above, then click Save all.
  3. Open your new bot in Telegram and send /start (or any message) — this lets the bot see your chat.
  4. Click Find my chat ID here. Pick your chat. The ID gets filled in automatically.
  5. Tick Send Telegram notifications and click Save all again.
  6. Click Send test message — you should see "MinerWatch · test" in Telegram within a second.

For a group: add the bot to the group, send any message, then use "Find my chat ID" — group IDs are negative numbers (e.g. -1001234567890).
